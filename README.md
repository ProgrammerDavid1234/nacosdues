# KDU NACOS Pay

A secure online payment gateway for KDU NACOS members. This is a 400 Level Computer Science Project with a focus on Human-Computer Interaction (HCI).

## Project Overview

KDU NACOS Pay simplifies the payment process for department dues, providing a seamless and secure experience for students.

## Technologies Used

This project is built with:

- **Frontend Framework**: React (with Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Icons**: Lucide React

## Getting Started

To get the project running locally:

1. **Clone the repository**
   ```bash
   git clone <YOUR_GIT_URL>
   cd nacos-pay-gateway
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

## Deployment

Build the application for production:

```bash
npm run build
```
